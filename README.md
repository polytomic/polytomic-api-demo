# Polytomic API Demo

## Getting Started

- Run `npm i` to install all dependencies
- Create a **.env** file in the root of the repo
- Add your *Polytomic API Key* to the **.env** file with the name *REACT_APP_PT_API_KEY*
    ```
    REACT_APP_PT_API_KEY='YOUR_POLYTOMIC_API_KEY'
    ```

- Run `npm run start`
